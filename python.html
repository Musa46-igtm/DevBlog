Python Programming

By Musa | 2025

Python is one of the most popular programming languages today, loved for its simplicity and versatility. Whether you want to automate tasks, build web apps, crunch data, or dive into AI — Python’s got your back.

🐍 Why Learn Python?

  • Easy to read and write, great for beginners and pros alike.
  • Massive ecosystem: Django, Flask, NumPy, pandas, TensorFlow, and more.
  • Used everywhere: web dev, data science, AI, automation, networking, even game dev.

🛠️ Setting Up

python --version
pip --version

Make sure you have Python and pip installed. You can get them from python.org.

✍️ Your First Python Script

print("Hello, world!")

Save it as hello.py and run with:

python hello.py

🔄 Variables & Data Types

name = "Musa"
age = 20
is_dev = True
scores = [90, 85, 78]

print(name, age, is_dev, scores)

🔁 Loops & Functions

def greet(user):
    print(f"Hello, {user}!")

for person in ["Amidah", "Salma", "John"]:
    greet(person)

⚙️ Popular Libraries

  • Django / Flask: Build powerful web apps & APIs.
  • pandas / NumPy: Data manipulation & math.
  • matplotlib: Charts & graphs.
  • TensorFlow / PyTorch: Machine learning & AI.
  • Requests: Make HTTP requests easily.

🚀 Example: Simple Flask App

# app.py
from flask import Flask
app = Flask(__name__)

@app.route("/")
def home():
    return "Hello from Flask!"

app.run(debug=True)

Run it with:

pip install flask
python app.py

🧑‍💻 Keep Learning

Explore automation with selenium or beautifulsoup4 for web scraping. Dive into data analysis with pandas or build neural networks with TensorFlow.

With Python, your only limit is your creativity. Start small, build cool projects, and level up step by step!
