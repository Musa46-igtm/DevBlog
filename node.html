Node.js is a powerful runtime built on Chrome's V8 JavaScript engine. It allows developers to use JavaScript for server-side programming, building scalable network applications with high performance.

ğŸ’¡ Simple HTTP Server

const http = require('http');

const server = http.createServer((req, res) => {
  res.writeHead(200, { "Content-Type": "text/plain" });
  res.end("Hello from Node.js!");
});

server.listen(3000, () => {
  console.log("Server running on http://localhost:3000");
});

Express.js makes building APIs with Node much simpler.

npm install express

const express = require('express');
const app = express();

app.get('/', (req, res) => {
  res.send('Hello World with Express!');
});

app.listen(3000, () => console.log('Server running on port 3000'));
